# **App Name**: ReThread

## Core Features:

- Item Listing: Users can upload clothing and accessories with title, description, category, condition, size, tags, location, and images.
- Browse and Search: Visitors can browse listings, search by keywords, filter by category, size, condition, price type, and distance.
- User Profiles: Registered users can create profiles with a public username, profile picture, city, and short bio.
- In-App Messaging: Embedded chat system tied to each item for registered users to communicate.
- Trust and Safety: Lightweight system with verified email badges and user reporting tools.
- Personalized Recommendations: AI tool to suggest relevant listings based on browsing history and saved searches.
- Local Proximity Sorting: Sort items based on the estimated geographic proximity to the user.

## Style Guidelines:

- Primary color: Soft beige (#F5F5DC) for a neutral and calming background.
- Background color: Light cream (#FFFDD0) maintaining a soft neutral tone.
- Accent color: Muted green (#A9BC8F) evoking sustainability.
- Headline font: 'Belleza' (sans-serif) for headings; Body font: 'Alegreya' (serif) for body text.
- Simple line icons to represent categories and actions.
- Clean, minimalist layout with large images and clear spacing.
- Subtle transitions for item loading and interaction feedback.
